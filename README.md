# Doctor Mario Clone

A mobile-optimized Doctor Mario clone built with Kaboom.js.

## Play the Game

You can play the latest version here:
- [GitHub Pages Demo](https://pablito-perez.github.io/doctor-mario-clone/)

## How to Test on Mobile

1. Open the [GitHub Pages Demo](https://pablito-perez.github.io/doctor-mario-clone/) in Safari.
2. Add it to your home screen for a more app-like experience:
   - Tap the "Share" button (square with arrow).
   - Select "Add to Home Screen".

## Controls

- **Left/Right Arrows:** Move the pill left or right.
- **Down Arrow:** Move the pill down faster.
- **Rotate Button:** Rotate the pill.

## Development

### Local Setup

1. Clone the repo:
   ```bash
   git clone https://github.com/pablito-perez/doctor-mario-clone.git
   ```
2. Open `index.html` in your browser.

### Tech Stack

- [Kaboom.js](https://kaboomjs.com/): Lightweight game library.
- HTML5 Canvas: Rendering.
- GitHub Pages: Hosting.
